# Week 3 Day 1 – Machine Learning Fundamentals
 Project Overview
This project demonstrates the complete Machine Learning workflow using a Student Performance Dataset. The objective is to predict student marks based on study hours, attendance, and assignments using Linear Regression.

##Dataset
Features:
* Study_Hours
* Attendance
* Assignments

Label:
* Marks
## Tasks Completed
### Dataset Exploration

* Loaded dataset using Pandas
* Viewed first and last records
* Checked shape, columns, and data types
* Generated summary statistics

### Machine Learning Workflow

 - Data Collection
 -Data Cleaning
 -Feature Selection
 -Train-Test Split
 -Model Training
 -Testing
 - Evaluation
 -Prediction

### Model Used

Linear Regression

### Evaluation Metrics

* Mean Absolute Error (MAE)
* R² Score

### Visualizations

1. Study Hours vs Marks
2. Attendance vs Marks
3. Assignments vs Marks
4. Regression Line Visualization
5. Predicted Marks Trend

### Predictions

Predictions were generated for new student records using the trained model.
